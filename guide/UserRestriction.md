# 用户限制

需要的权限：Device owner或Profile owner

Profile owner无法禁用部分功能，工作资料中部分功能无效，wearOS部分功能无效

功能前面的数字代表最少需要的API等级

打开开关就是禁用对应的功能，默认情况下所有开关都是关闭的

所有的用户限制都需要API24或以上
